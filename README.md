# 鬼滅の刃 ホームページ

Reactで作成された鬼滅の刃風のホームページです。

## 特徴

- モダンで美しいUIデザイン
- 鬼滅の刃のテーマに合わせたカラースキーム（赤、オレンジ、黒）
- レスポンシブデザイン対応
- スムーズなアニメーション効果

## セットアップ

### 依存関係のインストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

### ビルド

```bash
npm run build
```

### プレビュー

```bash
npm run preview
```

## 技術スタック

- React 18
- Vite
- CSS3 (カスタムスタイリング)

## セクション

- **ヒーローセクション**: メインタイトルと紹介
- **ストーリー**: 物語の概要
- **キャラクター**: 主要キャラクターの紹介
- **ギャラリー**: 画像ギャラリー（プレースホルダー）
